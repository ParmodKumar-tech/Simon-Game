# 🎮 Simon Game – Memory Challenge Game

## 📌 Overview

The Simon Game is a classic memory-based game designed to improve users' **memory retention and concentration**.

In this game:

* The system generates a sequence of colors
* The user must repeat the sequence correctly
* With each level, the sequence becomes longer and more challenging

---

## 🎯 Objective

* Improve memory retention
* Enhance focus and pattern recognition skills
* Provide an engaging and interactive experience

---

## 🛠️ Technologies Used

* **HTML** – Structure of the game
* **CSS** – Styling and visual design
* **JavaScript** – Game logic and interactivity

---

## 🚀 Key Features

* 🎨 Random color sequence generation
* 📈 Increasing difficulty with each level
* ❌ Game over detection on wrong input
* 🔁 Restart functionality

---

## 🧠 How It Works

1. Game starts when user presses a key/button
2. A random color flashes
3. User repeats the sequence
4. If correct → next level
5. If wrong → game over
